# Savitri Mart Website

A modern, responsive, and mobile-friendly website developed for Savitri Mart, Dhanera. The project is built using HTML, CSS, and JavaScript with a focus on performance, accessibility, and user experience.

## Features

- Responsive design for desktop, tablet, and mobile
- Product showcase
- Store gallery
- Contact section
- Google Maps integration
- WhatsApp and phone quick actions
- Progressive Web App (PWA) support
- Offline page using Service Worker
- SEO-friendly structure
- Clean and modern UI

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Font Awesome
- Progressive Web App (PWA)
